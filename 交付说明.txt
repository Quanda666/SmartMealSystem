================================================================================
一、项目概述
================================================================================

本项目是一个完整的智能配餐推荐系统，能够根据用户的个人特征、营养需求和口味
偏好，自动生成科学合理的每日餐单推荐。系统采用C++17标准开发，可在Visual 
Studio 2022中直接编译和运行。

================================================================================
二、功能实现情况
================================================================================

✅ 已完成所有题目要求的功能：

1. ✅ 用户口味偏好管理
   - 支持7种口味标签（清淡、甜、咸、酸、辣、鲜、香）
   - 可设置喜欢和避免的口味
   - 推荐时自动考虑口味偏好

2. ✅ 营养需求管理
   - 基于BMR/TDEE公式自动计算营养目标
   - 考虑年龄、性别、体重、身高、活动水平
   - 自动分配蛋白质、碳水化合物、脂肪比例

3. ✅ 历史记录功能
   - 保存所有餐单记录
   - 按日期查询历史
   - 统计分析历史数据

4. ✅ 每日餐单推荐
   - 自动生成早、中、晚三餐
   - 基于评分算法优化推荐
   - 确保营养均衡

5. ✅ 食物热量计算
   - 准确的热量计算
   - 宏量营养素计算
   - 实时总计显示

6. ✅ 营养成分可视化
   - 控制台进度条展示
   - 营养素占比显示
   - 目标达成度对比

7. ✅ 口味标签管理
   - 灵活的标签系统
   - 多标签支持
   - 动态调整

8. ✅ 手动调整推荐结果
   - 创建自定义餐单
   - 添加/删除食物
   - 评估算法匹配效果

三、技术实现
================================================================================

1. 编程语言：C++17标准
2. 开发工具：Visual Studio 2022
3. 构建系统：CMake 3.10+
4. 架构设计：分层架构 + 面向对象
5. 数据存储：文本文件持久化
6. 核心算法：基于评分的推荐算法

核心类：
- User（用户类）- 管理用户信息和营养目标
- Food（食物类）- 存储食物营养数据
- Meal（餐次类）- 管理单次餐食
- Database（数据库类）- 数据持久化管理
- RecommendationEngine（推荐引擎）- 实现推荐算法
- Utils（工具类）- 提供辅助功能

五、编译和运行
================================================================================

【Windows - Visual Studio 2022】

方法1：使用VS2022图形界面（推荐）
1. 打开Visual Studio 2022
2. 选择"打开本地文件夹"
3. 浏览到项目根目录
4. 等待CMake自动配置
5. 点击"生成" -> "生成全部"（Ctrl+Shift+B）
6. 点击"调试" -> "开始执行（不调试）"（Ctrl+F5）

方法2：使用批处理脚本
1. 双击 build.bat 进行编译
2. 双击 run.bat 运行程序

方法3：使用命令行
> mkdir build
> cd build
> cmake .. -G "Visual Studio 17 2022" -A x64
> cmake --build . --config Release
> .\bin\Release\MealRecommendationSystem.exe

【Linux】

$ ./build.sh
$ ./build/bin/MealRecommendationSystem

或手动编译：
$ mkdir build && cd build
$ cmake ..
$ make -j4
$ ./bin/MealRecommendationSystem

九、使用说明
================================================================================

快速开始：

1. 首次使用
   - 启动程序
   - 选择"2. 用户注册"
   - 输入个人信息
   - 系统自动计算营养目标

2. 获取推荐
   - 登录系统
   - 选择"1. 获取每日配餐推荐"
   - 查看推荐餐单
   - 选择是否保存

3. 设置偏好
   - 进入"7. 口味偏好设置"
   - 添加喜欢/避免的口味
   - 设置过敏源

4. 查看历史
   - 选择"2. 查看历史记录"
   - 查看历史餐单
   - 查看统计分析

详细使用说明请参考：docs/用户使用手册.md
