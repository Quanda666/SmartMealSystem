## 🚀 快速开始

### 1. 部署系统

```powershell
# 1. 克隆项目
git clone https://github.com/Quanda666/SmartMealSystem.git
cd SmartMealSystem

# 2. 检查环境
.\docs\scripts\check_environment.ps1

# 3. 编译项目（Visual Studio）
# 打开 MealRecommendationSystem.sln
# 选择 Release x64
# 按 Ctrl+Shift+B 编译

# 4. 部署
.\docs\scripts\deploy.bat

# 5. 验证部署
.\docs\scripts\verify_startup.bat
```

### 2. 设置自动备份

```powershell
# 创建自动备份任务（每天凌晨2点）
.\docs\scripts\create_backup_task.ps1
```

### 3. 系统监控

```powershell
# 实时监控系统状态
.\docs\scripts\monitor_system.ps1

# 运行性能测试
.\docs\scripts\performance_test.ps1
```

## 📝 使用文档中的脚本

### PowerShell脚本

所有PowerShell脚本都可以直接运行。首次运行需要设置执行策略：

```powershell
# 设置执行策略（首次运行，管理员权限）
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 运行脚本
.\脚本名称.ps1

# 查看脚本帮助
Get-Help .\脚本名称.ps1 -Detailed
```

### Batch脚本

```batch
REM 直接双击运行，或在命令提示符中执行
脚本名称.bat
```

### 管理员权限

某些脚本需要管理员权限：
- 右键点击脚本
- 选择"以管理员身份运行"

## 🔒 安全注意事项

### 数据安全
- ⚠️ 定期备份数据（建议每日自动备份）
- ⚠️ 测试备份的可恢复性
- ⚠️ 保留多个版本的备份

### 操作安全
- ⚠️ 生产环境操作前先在测试环境验证
- ⚠️ 重要操作前创建备份或还原点
- ⚠️ 记录所有重要操作

### 硬件安全
- ⚠️ 操作前断电并释放静电
- ⚠️ 避免在设备运行时进行硬件维护
- ⚠️ 使用正确的工具和方法
