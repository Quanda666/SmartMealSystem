## 如何使用文档中的脚本

### PowerShell脚本
```powershell
# 1. 设置执行策略（首次运行）
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 2. 运行脚本
.\脚本名称.ps1

# 3. 带参数运行
.\脚本名称.ps1 -Parameter Value
```

### Batch脚本
```batch
REM 直接双击运行，或在命令提示符中执行
脚本名称.bat
```

### 环境变量
某些脚本可能需要管理员权限，请右键"以管理员身份运行"。
