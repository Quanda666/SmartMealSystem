# 智能配餐推荐系统 - Web版

## 项目简介

这是一个基于C++17的智能配餐推荐系统，现在已经转换为Web版本，可以在Windows本地运行的Web图形化控制界面。

## 项目结构

```
MealRecommendationSystem/
├── MealRecommendationSystem.sln          # Visual Studio 2022 Solution文件
├── MealRecommendationSystem.vcxproj       # Visual Studio 2022 项目文件
├── src/                                   # 源代码目录
│   ├── main.cpp                          # 主程序（Web服务器）
│   ├── User.cpp                          # 用户类实现
│   ├── Food.cpp                          # 食物类实现
│   ├── Meal.cpp                          # 餐食类实现
│   ├── Database.cpp                      # 数据库操作实现
│   ├── RecommendationEngine.cpp          # 推荐引擎实现
│   ├── Utils.cpp                         # 工具函数实现
│   └── WebServer.cpp                     # Web服务器实现
├── include/                              # 头文件目录
│   ├── User.h                            # 用户类头文件
│   ├── Food.h                            # 食物类头文件
│   ├── Meal.h                            # 餐食类头文件
│   ├── Database.h                        # 数据库操作头文件
│   ├── RecommendationEngine.h            # 推荐引擎头文件
│   ├── Utils.h                           # 工具函数头文件
│   └── WebServer.h                       # Web服务器头文件
└── data/                                 # 数据目录
    └── .gitkeep                          # 数据文件占位符
```

## 功能特点

- **用户管理**：用户注册、登录、个人资料管理
- **食物库管理**：浏览、搜索食物数据库
- **智能推荐**：基于用户偏好和营养目标生成个性化餐食推荐
- **餐食记录**：记录和管理日常餐食
- **营养统计**：营养摄入分析和目标对比
- **Web界面**：现代化的Web图形化用户界面

## 如何在Visual Studio 2022中构建和运行

### 前提条件
- Visual Studio 2022 (版本17.0或更高)
- Windows 10/11

### 步骤

1. **打开Solution文件**
   - 双击 `MealRecommendationSystem.sln` 文件
   - 或在Visual Studio中选择"打开项目/解决方案"

2. **配置项目**
   - 选择构建配置（Debug 或 Release）
   - 选择目标平台（Win32 或 x64）

3. **构建项目**
   - 按 `Ctrl+Shift+B` 进行构建
   - 或在菜单中选择"生成" → "生成解决方案"

4. **运行程序**
   - 按 `F5` 开始调试运行
   - 或 `Ctrl+F5` 直接运行（不调试）

5. **访问Web界面**
   - 打开浏览器
   - 访问 `http://localhost:8080`
   - 开始使用智能配餐推荐系统

## 主要功能模块

### 1. 用户认证
- 用户注册：创建新账户
- 用户登录：验证用户身份
- 个人资料管理：维护用户信息

### 2. 食物数据库
- 食物信息管理：卡路里、蛋白质、碳水化合物、脂肪含量
- 食物标签系统：口味、类型分类
- 搜索和筛选功能

### 3. 智能推荐引擎
- 基于用户营养目标的个性化推荐
- 考虑用户口味偏好和过敏源
- 营养均衡的餐食组合建议

### 4. 餐食管理
- 日常餐食记录
- 餐食历史查看
- 编辑和删除餐食记录

### 5. 数据统计
- 营养摄入统计
- 目标达成分析
- 趋势分析图表

## 开发说明

### 添加新食物
程序首次运行时会自动创建示例食物数据。您可以：
1. 通过Web界面的"食物库"页面添加
2. 直接编辑 `data/foods.txt` 文件

### 自定义营养目标
用户注册后可设置：
- 基本信息：年龄、性别、身高、体重
- 活动水平：久坐、轻度活动、中度活动、高强度活动
- 系统会自动计算每日营养需求

## 故障排除

### 端口占用
如果8080端口被占用：
1. 修改 `src/main.cpp` 中的端口号
2. 重新编译运行

### 数据文件损坏
如果数据文件损坏：
1. 删除 `data/` 目录下的所有文件
2. 重新运行程序，会自动创建示例数据

### 编译错误
如果遇到编译错误：
1. 确保使用Visual Studio 2022
2. 检查是否安装了Windows SDK
3. 确认项目配置正确
