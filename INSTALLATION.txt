智能配餐推荐系统 - 安装和运行说明
===========================================

一、系统要求
-----------
1. Windows 10/11 或 Linux系统
2. Visual Studio 2022 (Windows) 或 GCC 8.0+ (Linux)
3. CMake 3.10或更高版本
4. 至少512MB内存
5. 至少50MB硬盘空间

二、Windows (VS2022) 快速安装
-----------------------------
方法1：使用Visual Studio 2022（推荐）
1. 打开Visual Studio 2022
2. 选择"打开本地文件夹"
3. 浏览并选择项目根目录
4. 等待CMake自动配置
5. 点击"生成" -> "生成全部"（或按Ctrl+Shift+B）
6. 点击"调试" -> "开始执行（不调试）"（或按Ctrl+F5）

方法2：使用批处理脚本
1. 双击 build.bat 进行编译
2. 双击 run.bat 运行程序

三、Linux 快速安装
-----------------
1. 打开终端
2. 进入项目目录：cd /path/to/MealRecommendationSystem
3. 运行编译脚本：./build.sh
4. 运行程序：./build/bin/MealRecommendationSystem

四、首次使用
-----------
1. 启动程序后选择"2. 用户注册"
2. 输入用户名和密码
3. 填写个人信息（年龄、体重、身高等）
4. 系统自动计算每日营养目标
5. 返回主菜单，选择"1. 用户登录"
6. 输入注册的用户名和密码
7. 选择"1. 获取每日配餐推荐"开始使用

五、常见问题
-----------
Q1: 编译错误：找不到CMake
A1: 请安装CMake 3.10或更高版本

Q2: 中文显示乱码
A2: Windows用户在cmd中执行：chcp 65001
    Linux用户执行：export LANG=zh_CN.UTF-8

Q3: 找不到data文件夹
A3: 程序会自动创建data文件夹并初始化数据

Q4: 如何备份数据
A4: 复制data文件夹到其他位置即可
